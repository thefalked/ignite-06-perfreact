# Optimizing Performance on React

This is a [Next.js](https://nextjs.org/) project.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

and the json server:

```bash
npm run server
# or
yarn server
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
